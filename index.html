Formdata

获取表单（对象）下的dom 值 并构造为 json 格式，或者赋值表单（对象）下的 dom 初始值

View on GitHub Download .zip Download .tar.gz

这是一个简单获取表单数据的插件，它依赖 requireJS ,jquery ，它支持不是 from 的任何对象下包含的所有dom的值

获取表单数据 form.getData("temp")

赋值显示数据
form.initData("temp", { "name": "123" });
