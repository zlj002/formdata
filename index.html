Formdata

获取表单（对象）下的dom 值 并构造为 json 格式，或者赋值表单（对象）下的 dom 初始值

View on GitHub Download .zip Download .tar.gz

这是一个简单获取表单数据的插件，它依赖 requireJS ,jquery ，它支持不是 from 的任何对象下包含的所有dom的值 假如你的html 代码如下

<div id="temp"> <input type="text" name="name"/><br /> <input type="date" name="brithday" /><br /> <label><input type="radio" name="sex" value="1" />男</label> <label><input type="radio" name="sex" value="0" />女</label> <br /> <label> <input type="checkbox" name="interest" value="1" />运动</label> <label> <input type="checkbox" name="interest" value="2" />休闲</label> <br /> <input type="text" name="detail.name"/> <br /> <input type="text" name="a.b.name" /> <br /> <img name="headerImage" src="http://developer.baidu.com/resources/online/common/img/logo.png"/> <br /> <input type="button" id="btnGetData" value="获取数据"/> <input type="button" id="btnInitData" value="赋值" /> </div>

获取其中的数据数据

$("#btnGetData").on("click", function () { alert(JSON.stringify(form.getData("temp"))); });

赋值显示 $("#btnInitData").on("click", function () { var data = { "name": "123", "sex": "0", "interest": "2", "detail": { "name": "test" }, "headerImage": "http://developer.baidu.com/resources/online/errors/img/img_404.png", "a": {"b":"123123"},"brithday":"2015-02-11" }; form.initData("temp", data); });
